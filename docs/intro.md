Let's get started with **sbom.cc**, *SBOM Parsing for Humans*.

## Parse a local SBOM file

### What you'll need

sbom.cc runs on modern versions of Linux and macOS.

Tested on:

- macOS 14 (Apple M1/M2)
- Ubuntu 22.04.02 (Linux 5.15)
